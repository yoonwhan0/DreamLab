## 정체성 (2026-07)

### 홈 히어로

> **같은 꿈을 꾼 사람들.**  
> **30일 뒤, 그들에게는 무슨 일이 있었을까요?**

### 「우리는 어떤 것을 연구하는가」(아코디언)

히어로 바로 아래 클릭형 텍스트 — 펼치면:

1. AI가 꿈 해몽을 해주는 건, 이제 일반적입니다.
2. 그 기록은 쌓입니다. 웃고 넘기기엔, 너무 많이 모이고 있습니다.
3. **우리는 생각했습니다. 해몽이 끝난 뒤, 그다음은 무엇일까?**

→ 연구 축 3개 (30일 뒤 / 쌓인 패턴 / 해몽 vs 현실)

### 톤 가이드

- ✅ 관측, 기록, 30일, 데이터, 결말, 연구소
- ✅ 담백·호기심 — FOMO·과한 블러 지양 (2026-07 UX 정리)
- ❌ 신비·예언·불길 단정
- ❌ 「별일 없었음」 후기 카테고리 (→ `other` 정규화)

## 제품 구조

```
[입구] 꿈 기록 + AI 해몽 — 익명 Auth로도 저장
    ↓
[30일] 푸시 → 후기 작성 (회원)
    ↓
[탐색] 같은 꿈 N명 통계 · 후기 (시드 DB + 실유저)
    ↓
[프리미엄] 8주 운세 7축 그래프 · 후기 전체 (스토어 IAP 예정)
```

## 경쟁력

| 일반 해몽 | DreamLab |
|-----------|----------|
| GPT 1회 답변 | **30일 후 실제 후기 DB** + 시드 100건+ |
| 예언 | 통계 경향 + 면책 |
| AI 후기만 | **실데이터 ≥5건** 시 합성 대신 Firestore |
| 웹 결제 | **토스 제거** → App Store / Play |

## 사용자 여정

1. **스플래시** — DreamLab (~1.8초)
2. **홈** — 히어로 · 연구 미션 · 관측 밀도 · 키워드 칩
3. **꿈 기록** — `/write`
4. **AI 해석** — `POST /api/interpret-dream`
5. **30일 후** — `/follow-up/:id`
6. **탐색** — 키워드별 후기·운세
