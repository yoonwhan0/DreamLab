## 정체성 (2026-07)

### 메인 슬로건

> **우리는 꿈의 결과가 궁금했습니다.**

### 서브 (홈 히어로, 3줄)

1. 해몽이 닿지 못한 금단의 영역 —
2. 당신만 아직 모르는 결말 —
3. **같은 꿈을 꾼 이들의 한 달 뒤, 지금 열립니다.** *(강조)*

### 매니페스토

> 꿈의 결말은 금기였습니다. 혼자 해석하면 끝났고, 아무도 한 달 뒤를 말해주지 않았습니다. 우리는 그 영역을 열었습니다.

### 톤 가이드

- ✅ 관측, 기록, 30일, 데이터, 결말, 연구소
- ✅ 담백·호기심·금단(브랜드 분위기)
- ❌ 버튼에 「금단 입장」 같은 모호한 CTA
- ❌ 신비·예언·불길 단정
- ❌ 영문 브랜드를 CSS `uppercase`로 강제 변환

## 제품 구조

```
[입구] 꿈 기록 + AI 관측 메모 — 비회원도 DB 저장 (익명 Auth)
    ↓
[회원] 유사 꿈 · 30일 푸시 · 후기 작성
    ↓
[프리미엄] 한 달 뒤 통계·후기 전체 열람 — ₩4,900/월 (결제 연동 전)
```

## 경쟁력

| 일반 해몽 | DreamLab |
|-----------|----------|
| usualTake 한 줄 | **30일 후 실제 후기 DB** |
| GPT 1회 답변 | 같은 꿈 N명 + 답변율 % |
| 예언 | 통계 경향 + 면책 (`LEGAL_DISCLAIMER`) |
| 고정 키워드 사전 | **AI `researchAnchor`** 1차 + 코드 폴백 |

## 사용자 여정

1. **스플래시** — DreamLab / 꿈연구소, LOADING (~1.8초)
2. **꿈 기록** — `/write` (하단 탭 두 번째)
3. **AI 해석** — `POST /api/interpret-dream`
4. **30일 대기** — Cloud Functions 푸시 예약
5. **결과 기록** — `/follow-up/:id` (회원)
6. **패턴 비교** — 탐색·프리미엄 통계

## Admin (운영)

- 별도 앱 `admin/` — 사용자 앱에 링크 없음
- Firestore `users/{uid}.role = "admin"` 으로 접근
- 홈 KPI·데이터 노출·푸시 문구 등 **`config/*` 문서** — 재배포 없이 변경
